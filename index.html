Homepage

This RestAPI consists of the following endpoints

  • /students
  • /items

Students Endpoint

This endpoints accepts a GET request and returns a list of students. The format of the response body is given below

[ { "name": "Benjamin", "seat": 2, "class": "B" }, ... ]


This endpoints also accepts a POST request and returns a list of students which includes the newly saved information. The format of the request body is given below

{ "name": "Benjamin", "seat": 2, "class": "B" }

Items Endpoint

This endpoints accepts a GET request and returns a list of items. The format of the response body is given below

[ { "id": 1, "name": "Airpods", "price": 12500 }, ... ]
